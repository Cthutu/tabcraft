# TabCraft

TabCraft is a single HTML file that contains a fully functional browser bookmark
manager.  It is designed to be simple, fast, and easy to use.  It is also
designed to be used as your browser's homepage, so that you can access your
bookmarks quickly and easily.

## Features

- **Single HTML file**: TabCraft is a single HTML file that can be easily
  downloaded and used as your browser's homepage.

- **Fully functional**: TabCraft is a fully functional bookmark manager that
  allows you to add, edit, and delete bookmarks.  You can also organize your
  bookmarks into folders.  It uses the browser's local storage to save your
  bookmarks, so they will be available even when you are offline.

- **Customizable**: TabCraft is highly customizable.  You can change the
  appearance of the page by editing the CSS, and you can add new features by
  editing the JavaScript.

- **Keyboard shortcuts**: TabCraft supports keyboard shortcuts for common
  actions, such as adding a new bookmark, opening a bookmark, and navigating
  between folders.  Keys such as `/` to search, `n` to add a new bookmark, and
  `j` and `k` to navigate between bookmarks etc.

- **Import/Export**: TabCraft allows you to import and export your bookmarks in
  JSON format, so you can easily back up your bookmarks or transfer them to
  another browser, and also to sync them across devices using a cloud storage
  service like Dropbox, Pastebin, etc.  This will manage merges and conflicts
  when importing bookmarks, so you can easily keep your bookmarks up to date
  across all your devices.

## Design

On the left side of the page, there is a sidebar that contains a list of your
folders.  You can click on a folder to view the bookmarks inside it.  You can
also click on the "Add Folder" button to create a new folder, right-click on a
folder to rename or delete it, and drag and drop folders to reorder them.

On the right side of the page, there is a main area that displays the bookmarks
in the selected folder.  You can click on a bookmark to open it replacing the
current page, or right-click on a bookmark to edit or delete it.  You can also
click on the "Add Bookmark" button to create a new bookmark in the selected
folder.  Additionally, you can drag and drop bookmarks to reorder them within a
folder, or to move them to a different folder.

The bookmarks appear as cards with the bookmark's title and favicon.  The cards
are arranged in a grid layout, and they are responsive, so they will adjust to
the size of the screen.  The page also has a search bar at the top that allows
you to quickly find bookmarks by title.

## Usage

To use TabCraft, simply download the `index.html` file and open it in your
browser.  You can then start adding your bookmarks and organizing them into
folders.  You can also customize the appearance and functionality of the page by
editing the HTML, CSS, and JavaScript.  Furthermore, you can use any text editor
to edit the file, and you can use the browser's developer tools to inspect and
debug the code.  You can also share the `index.html` file with others, so they
can use it as well.
